# codeandcreative

A portfolio website

Built on webpack-starter-html:

* webpack 4
* babel 7
* html-webpack-plugin
* sass loading
* [scssstarter](https://github.com/johnfyoung/scssstarter)
* webpack-dev-server

## Install

To install, clone this repo and type `yarn`.

To run the dev server, type `yarn start`.
